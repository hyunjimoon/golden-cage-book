## 핵심 방정식

```
dG/dF = (dG/dR) × (dR/dF) = (+) × (−) = (−)
```

자원(F)이 방향전환(R)을 억압하고, 방향전환이 성장(G)을 만든다. 168,011개 미국 벤처 실증.
