# 답과 질문

## 답

네 사람이 하나씩 빌렸고, 하나씩 잃었다.

| 인물   | 빌린 것       | 잃은 것       |
| :--- | :--------- | :--------- |
| 엠마   | 욕망         | 삶          |
| 살리나  | 품위         | 시간         |
| 파늘루  | 확신         | 목숨         |
| 로빈슨  | 질서         | 자유         |
| 방드르디 | 아무것도 안 빌린다 | 아무것도 안 잃는다 |

빌린 것을 안 갚으면, 자기 것을 잃는다.

그리고 빚은 돈다. 욕망을 빌리면 품위를 빌려야 하고, 품위를 빌리면 확신을 빌리게 되고, 확신을 빌리면 질서를 빌려 세우고, 질서를 빌리면 장부를 열 수 없다. 새장은 혼자 완성되지 않는다 — 네 빚이 서로를 짓는다.

답은 네 동사다. 장부를 펴라. 순서를 타라. 확신을 교정하라. 다음 불을 피워라. 빌린 것을 갚으면 새장이 둥지가 된다.
