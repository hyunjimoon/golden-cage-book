# LEXICON: 황금 새장을 열다 — Semantics

> 이 책에서만 쓰이는 알파벳. 문체템플릿이 이 알파벳의 **문법**(syntax)이라면, 이 파일은 **뜻**(semantics).
> 편집자가 원고를 읽을 때, 독자가 책을 읽은 후 참조.

| révolte = 반항         | 카뮈: debt을 인정하되 새 위안으로 재융자하지 않겠다는 선언. 3S Framework의 경영학적 실현. 실패=재융자(다시 패치), 생존=반항(빚 인정 후 구조 변경)                                                                    | 2026-03-13 |
| 빈손이되 맨손이 아니다         | 4장 선언문. 빈손=결과물 없음(외부 자원 사라짐). 맨손=도구 없음(변신 능력 없음). 4번의 변신 근육(이해·행동·책임·장난)은 남아있다. Funding이 사라져도 Reposition 능력은 살아있는 상태                                              | 2026-03-13 |
| 4-근육 변신 아크           | 1장 +이해(무모→용기) → 2장 +행동(냉소→지혜) → 3장 +책임(신념→책임) → 4장 +장난(균형→새균형). 누적적(이전 근육이 전제조건), 순환적(과잉이 다음 새장), 4장에서 축 전환(메타-변신)                                                | 2026-03-14 |
| 장난(作亂)               | 4장 근육의 이름. "작란"=일을 벌인다. 무질서가 아니라 의도적 해체. 기존 질서를 부수되 새 질서로 가는 것. S커브의 변곡점에서 일어나는 행위                                                                                | 2026-03-14 |
| 신념 ≠ 믿음              | 이 책에서 "신념"으로 통일. 신념윤리(Gesinnungsethik)가 Weber의 고정 학술용어이므로. 파늘루=신념윤리(원칙의 정당성), 리외=책임윤리(결과에 대한 응답)                                                                  | 2026-03-14 |
| 품질 = fitness of use    | Juran의 정의. 좋은 것이 아니라 **쓰임에 맞는 것**. 다이어그램의 품질 = 핵심 독자(창업 고민 대학생 + 부모)가 한 눈에 이해하고 행동을 바꾸는가. 학술적 정확성이 아니라 행동 전환 확률 / decode cost. 소통밀도의 시각적 버전. 결기에 찬, 비장한 느낌 = 톤의 fitness | 2026-03-15 |
| 빌린 것을 갚다 | 빌린 것을 안 갚으면, 자기 것을 잃는다. "갚다"는 돌려주는 게 아니라 장부에서 지우는 것. "바꾸다"가 아닌 "갚다" — 빌리다/빚/갚다/장부가 하나의 재무 은유로 관통 | 2026-03-19 |
| 빚 = 간극 / 노력 | **빚은 넘어짐이 아니라, 일어서기를 멈춤이다.** 간극=약속과 현실의 거리. 노력=간극을 줄이려는 행동(장부를 펴라·순서를 타라·확신을 교정하라·다음 불을 피워라). 노력이 멈추면 빚이 쌓이고, 노력이 지속되면 빚이 줄어 새장이 둥지가 된다. 엠마: 간극 큼, 노력 없음(장부 안 폄)→빚 폭발. 살리나: 간극 큼, 노력 없음(73년 멈춤)→빚 폭발. 파늘루: 간극 큼, 노력 없음(모델 안 고침)→빚 폭발. 리외: 간극 큼, 노력 지속(매일 교정)→빚 청산. 방드르디: 간극 없음(빌린 게 없음)→빚 없음. 출처: [[죄의 정의]] | 2026-04-06 |
| 새장 → 둥지 | 빌린 것을 갚으면 구조는 사라지지 않는다 — 새장이 둥지가 된다. 새장과 둥지는 같은 구조. 차이는 빌린 것이냐 내 것이냐 | 2026-03-19 |
| 신념 ≠ 확신 | 신념(conviction) = 내용, 바뀔 수 있다. 확신(certainty) = 메타 수준의 요구, "이것이 맞아야 한다." 파늘루의 신념(심판→고통)은 바뀌었으나, 확신(tout ou rien)은 안 바뀜. Bayesian calibration의 적 = certainty(uncertainty의 부정) | 2026-03-19 |
| 4공간 (순환) | 무도회장(1→2, 피치데이) → 항구(2→3, 첫 고객) → 성벽(3→4, 스케일링/고착) → 거울(4→1, 혼자 서기). 장은 진단, 공간은 경험. dG/dF=(−)가 극대인 지점 = 성벽(SW) | 2026-03-19 |
| 다/세 진화 | 다: 엠마→타루("나는 보균자였다"를 인정). 세: 살리나→리외(60%인 채로 주사기를 든다). 다+세 = 방드르디를 향한다 — 분석과 행동이 분리되지 않는 상태 = 빚=0 = 계산끝낸물 | 2026-03-19 |
| 기세(Schwung) | 2장 철학 핵심. 클라우제비츠: 안개 속에서도 멈추지 않는 힘. 완벽한 지도를 기다리면 기세가 죽고 군대가 죽는다. 살리나=기세 없음(관조의 감옥), 탄크레디=기세 있음(60%의 지도로 뛴다). 행동 속도의 원료 | 2026-04-01 |
| 소진(인공선택의 소진) | 4장 철학 핵심. 성공한 선택이 변이를 굶겨서, 자기가 작동할 재료를 없앤다. 로빈슨의 시계가 파도를 죽인다. S커브 후반의 병: 정밀한 관리가 가능성을 굶기는 역설. 경로고착의 생물학적 이름 | 2026-04-01 |
| 전역최적 vs 지역최적 | 3장 실천 핵심. 각 팀의 100%=파늘루(지역최적 — 내 팀의 목적함수에 충실). 두 팀의 f를 합쳐서 보는 것=리외(전역최적 — 전체 목적함수). 가중치를 결과를 보고 수정하는 것이 책임. "영웅이 되려 하지 마라. 의사가 되어라"의 수학적 번역 | 2026-04-01 |
| Nail↔Scale | 2장 실천의 두 함정. 살리나 마비(Nail→Scale 실패)=봤는데 안 뜀. 보바리 조산(Nail 전에 Scale)=안 봤는데 뜀. Nail=문제를 정확히 못 박는 것. Scale=못 박은 것을 확장하는 것. 순서가 뒤집히면 빚 | 2026-04-01 |

## 진행자: 다(DA) / 세(SE)

> Dimo Dimov의 Bermuda Quadrilateral에서 추출.

| | 다(DA) | 세(SE) |
|:--|:--|:--|
| **약자** | **D**esigner + **A**rtist | **S**cientist + **E**ngineer |
| **기능** | Vision + Value | Understanding + Solution |
| **ISRV** | 수면(보여주는 자): V→R'→S'→I' | 물밑(짓는 자): I→S→R→V |
| **기본 역할** | 소설 서술, 감각 묘사, 독자 대리 질문 | 이론 정의, 수식, 데이터 분석 |
| **역전 시** | 이론을 발견한다 (Scientist 영역 침범) | 감정을 고백한다 (Artist 영역 침범) |

**역전이 이름에 내장된다**: 다(DA)가 이론을 발견하는 순간 = D/A가 S 영역으로 넘어간다. 세(SE)가 감정을 고백하는 순간 = S/E가 A 영역으로 넘어간다. 이름의 경계를 본문이 깨는 구조 — 이것이 호감의 메커니즘.

**역전 비율의 궤적**: 세→다 역전 비율이 장마다 증가한다 (1장 1회 → 2장 2회 → 3장 3회 → 4장 5회). 4장에서 역전이 역전된다 — 세 자신이 변신의 대상이 됨.

## 순환 구조: 질문과답 → 4장 → 답과질문

| 위치 | 파일 | 기능 |
|:--|:--|:--|
| 프롤로그 | [[0_질문과답]] | 4개 질문 → 잠정적 답 → "1장을 펼쳐보라" |
| 본문 | 1~4장 | 각 장이 하나의 새장을 열며 근육 하나를 얻는다 |
| 에필로그 | [[5_답과질문]] | 4개 답(근육) → 각 답이 낳는 새 질문 → "0장으로 돌아가라" |

답이 새 질문을 낳는 순환:
- 이해를 얻었다 → 잘 보는 눈이 발을 묶는다 → 왜 못 움직이는가?
- 행동을 얻었다 → 빠른 발이 확신을 낳는다 → 내 확신이 누구의 비용인가?
- 책임을 얻었다 → 매일의 판단이 질서가 된다 → 내가 만든 것이 나를 가두는가?
- 장난을 얻었다 → 빈손은 장부를 안 편다 → 이 욕망은 어디서 왔는가?

마지막 질문 = 첫 질문. dG/dF=(−)의 서사적 순환.
